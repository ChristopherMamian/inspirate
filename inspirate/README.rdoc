== README

How to run this app

* git clone into a local directory

* cd into repository

* Run <tt>bundle install</tt>

* Run <tt>bundle exec rake db:create</tt>

* Run <tt>bundle exec rake db:migrate</tt>

* Run <tt>rails server</tt>

* Go to <tt>http://localhost:3000/</tt> via your browser.
